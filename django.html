模板的'.'的用法:
  1. 引用字典 
    from django.template import Template, Context
person = {'name': 'Sally', 'age': '43'}
t = Template('{{ person.name }} is {{ person.age }} years old.')
c = Context({'person': person})
t.render(c)
#result is: u'Sally is 43 years old.'
  2. 引用list的索引 
    from django.template import Template, Context
t = Template('Item 2 is {{ items.2 }}.')
c = Context({'items': ['apples', 'bananas', 'carrots']})
t.render(c)
# result is :u'Item 2 is carrots.'
  3. 引用对象的属性 
     from django.template import Template, Context
 import datetime
 d = datetime.date(1993, 5, 2)
 t = Template('The month is {{ date.month }} and the year is {{ date.year }}.')
 c = Context({'date': d})
 t.render(c)
# result is : u'The month is 5 and the year is 1993.'
  4. 引用对象的方法(只能引用不需要参数的方法) 和引用对象的属性类似，不需要()

  • addslashes: 添加反斜杠到任何反斜杠
  • date: 按指定的格式字符串参数格式化 date 或者 datetime 对象，范例： {{ pubdate|date:"F j, Y" }}

  1. 创建一个模板对象
  2. 创建一个contex对象(和字典相似)
  3. render模板 
from django import template
t = template.Template('My name is {{ name }}.')

因为是重复性的工作，所以django提供了shortcut,建议现在settings.py设定 TEMPLATEDIRS 

from django.shortcuts import render_to_response
import datetime

def current_datetime(request):
    now = datetime.datetime.now()
    return render_to_response('current_datetime.html', {'current_date': now})

rendertoresponse只需要提供模板文件，与一个字典，它会渲染模板，并且返回一 个HttpResponse对象,极大的简化了工作

模板的包含与继承

  • include标签 {% include "includes/nav.html" %}
  • 模板继承 base.html: 
    <!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN">
<html lang="en">
  <head>
    <title>{% block title %}{% endblock %}</title>
  </head>
  <body>
    <h1>My helpful timestamp site</h1>
    {% block content %}{% endblock %}
    {% block footer %}
    <hr>
    <p>Thanks for visiting my site.</p>
    {% endblock %}
  </body>
</html>

    {% extends "base.html" %}

{% block title %}The current time{% endblock %}

{% block content %}
<p>It is now {{ current_date }}.</p>
{% endblock %}

    每一个block就是一个子模板可以重定义的区域 根据需要使用任意多的继承次数。使用继承的一种常见方式是下面的三层法：

    1. 创建 base.html 模板，在其中定义站点的主要外观感受。这些都是不常修改甚至从不修改的部分.
    2. 为网站的每个区域创建 baseSECTION.html 模板(例如, basephotos.html 和 baseforum.html )。这些模板对 base.html 进行拓展，并包含区域特定的风格与设计。
    3. 为每种类型的页面创建独立的模板，例如论坛页面或者图片库。这些模板拓展相应的区域模板。

Module

  1. 第一步必须设置settings.py的数据库相关域,用以下代码验证配置是否正确： 
    python manage.py shell #enter django shell

from django.db import connection
cursor = connection.cursor()
  2. 创建一个app(使用module必须创建app) python manage.py startapp appname 然后在module.py添加代码 
    from django.db import models

# Create your models here.
class Publisher(models.Module):
    """
    """
    name = models.CharField(max_length = 30)
    address = models.CharField(max_length = 50)
    city = models.CharField(max_length = 60)
    state_province = models.CharField(max_length = 30)
    country = models.CharField(max_length = 50)
    website = models.URLField()
class Author(models.Model):
    first_name = models.CharField(max_length=30)
    last_name = models.CharField(max_length=40)
    email = models.EmailField()

class Book(models.Model):
    title = models.CharField(max_length=100)
    authors = models.ManyToManyField(Author)
    publisher = models.ForeignKey(Publisher)
    publication_date = models.DateField()

    然后运行python manage.py validate检查模型语法是否正确，无误后运行 python manage.py sqlall books会显示最终运行的sql语句 python manage.py syncdb会最终执行数据库的创建

  3. 访问修改数据库 
    # insert a record
p = Publisher(name='Apress',
        address='2855 Telegraph Ave.',
        city='Berkeley',
        state_province='CA',
        country='U.S.A.',
        website='http://www.apress.com/')
p.save()

# just update, do not create record
p.name = 'Apress Publishing'
p.save()

Publisher.objects.filter(id=52).update(name='Apress Publishing')
# 访问数据，转换为select语句
Publisher.objects.all()
Publisher.objects.filter(country="U.S.A.", state_province="CA")
# select ... order by ...
Publisher.objects.order_by("name")
#逆序
Publisher.objects.order_by("-name")
#连锁查询，同时指定where与order by子句
Publisher.objects.filter(country="U.S.A.").order_by("-name")
# select ... order by ... limit ...
Publisher.objects.order_by('name')[0]
Publisher.objects.order_by('name')[0:2]

#delete record
Publisher.objects.filter(country='USA').delete()
Publisher.objects.all().delete()

表单

request.path 除域名以外的请求路径，以正斜杠开头 "hello"
request.gethost() 主机名（比如，通常所说的域名） "127.0.0.1:8000" or "www.example.
request.getfullpath() 请求路径，可能包含查询字符串 "hello?print=true"
request.issecure() 如果通过HTTPS访问，则此方法返回True，否则返回False True 或者 False  
request.META 一个字典，包含很多客户端信息(key: HTTPREFERER, HTTPUSERAGENT) request.META.get('HTTPUSERAGENT', 'unknown')
request.GET 表单的get方法传递的数据(是一个类字典结构) request.GET['q']就是查询字符串
request.POST 表单的post方法传递的数据（是一个类字典结构）
