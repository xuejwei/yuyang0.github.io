1 evil-mode

1.1 常用的命令：

  1. w 下一个单词, b 前一个单词
  2. r, R: r修改当个字符，R则会进入一种overwrite模式，可以替换从当前光标开始的任何 字符
  3. c：可以修改当前光标所在的字符，词以及行，c, cw, c$，这些命令的前面可以添加数字
  4. d: 删除，dd, dw, d$, 可以在前面添加数字
  5. u: undo， Ctrl-R 是redo
  6. :!command: 执行外部命令， :!ls
  7. 搜索：/somepattern ENTER，如国反向那么就用？，在搜索模式下n代表下一个匹配，N 代表前一个匹配
  8. 替换：
    • :s/old/new: 替换本行的第一个old为new
    • :s/old/new/g: 替换本行的所有old为new
    • :%s/old/new/g: 替换所以的old为new
    • :2,20s/old/new/g: 替换第2行到第20行之间的old为new
  9. .: 重复上一条命令

1.2 visual mode

  1. v(字符可视化), V(行可视化) C-v(列可视化)
  2. vi{,vi[, vi(, vi<, vis, vip: 选中{}, [], (), <>, 句子，段落。
  3. d/D: 删除选中的文本，前者仅仅只删除选中的文本，而后者会删除所有选中文本所在 的行，即便该行只有一部分文本被选中
  4. y/Y: 复制选中的文本，后者会复制所有的行
  5. c/C: 删除所有选中的文本同时进入编辑模式。
  6. <, >: 选中的字符左移或者右移shiftwidth个字符
  7. u/U/~: 选中的文本全部变小写/变大写/反转大小写。
  8. I: 所有行的开头插入
  9. A: 在所有行的末尾插入
  10. J: 将所有选中的行用空格连接起来

2 spacemacs键绑定

2.1 文本相关(SPC x)

  1. SPC x u: 选中的部分小写
  2. SPC x U: 选中的部分大写
  3. SPC x m j: 当前行向下移动
  4. SPC x m k: 当前行向上移动
  5. SPC x d w: 删除所有行的末尾的空格

2.2 文件相关(SPC f)

  1. SPC f f: ido open
  2. SPC f F: helm open
  3. SPC f r: open a recent file
  4. SPC f o: open a file with external program
  5. SPC f s: save current file
  6. SPC f S: save all files

2.3 buffer相关(SPC b)

  1. SPC b d: delete current buffer
  2. SPC b m h/j/k/l : move to left/below/above/right buffer
  3. SPC b s: 用helm来切换buffer，类似于C-x b
  4. SPC b r: 重命名当前buffer

2.4 Window相关(SPC w)

  1. SPC w v/V: 纵向分割窗口
  2. SPC w s/S: 横向分割窗口
  3. SPC w h/H: 跳到 左边/最左边 的窗口
  4. SPC w j/J
  5. SPC w k/K
  6. SPC w l/L
  7. SPC w c: 关闭当前窗口
  8. SPC w m: 最大化或者最小化当前窗口
  9. SPC w 0/1/2/../9:　每一个窗口都有编号，该命令跳转到编号的窗口

2.5 ace-jump

  1. SPC SPC: ace word mode
  2. SPC l: ace line mode

2.6 expand-region

  1. SPC v：接着按ｖ就可以不断的扩大选择区域, 按V可以缩小区域

2.7 NeoTree

  1. SPC ft: 打开NeoTree

2.8 surroud

  1. cs[': 将[换成',在normal state可以用
  2. s "： 需要先选定文本，然后才能包裹

2.9 search(SPC s)

  1. SPC s /: 用外部程序来搜索

2.10 Line

  1. J: 将当前行和下一行连接起来。
  2. SPC j j: 将当前行从光标处分成两行

2.11 Comment

  1. SPC c l: 注释当前行，如果要注释多行，那么在命令前加数字

2.12 smartparens

  1. spc k ` s: 右括号向右吞并一个元素
  2. spc k b: 右括号向左吐出一个元素

2.13 其他：

  1. SPC i j: 在当前行的下面插入一个空行
  2. SPC i k: 在当前行的上面插入一个空行
  3. >>: indent当前行，5>> indent下面的5行
  4. <<: 和上面类似
