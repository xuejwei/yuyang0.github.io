一些惯例以及基本常识

  1. 在jquery中 \( 就是jQuery的简写,故\).trim 就是jQuery.trim的等价表示
  2. \((function(...){...})是\)(document).ready(function(…){…})的简写
  3. 给超链接添加click事件时,应该返回false,因为这样浏览器会认为用户没有点击超链接,从而阻止浏览器跳转

  4. DOM元素与jquery元素(this是DOM对象, $(this)是jquery对象)前者是html文档的 DOM树中的一个节点,可以使用getElementById得到,DOM对象可以使用 javascript 的方法,比如innerHTML, 后者是jquery包装DOM元素后产生的对象, 只能使用 jquery的方法比如.html(), .text()

    二者之间的相互转换:

    1. DOM => jquery : $jqueryObj = $(DOMObj) 直接用$包裹DOM对象即可转换为 jquery对象
    2. query=> DOM : DOMObj = $jqueryObj[0], jquery对象是一个数组对象,一个 jquery对象可以包含多个DOM对象,所以可以通过index或者get(idx)来获得DOM 对象

jquery selector

基本选择器

jQuery的基本语法是：$(selector).action(), 常用的选择器有：

Table 1: 基本选择器
选择器 描述 返回 示例
#id 根据指定id匹配一个元素 单个元素 $("#test")
.class 根据指定的类名匹配元素 集合元素 $(".test")
element 根据元素名匹配元素 集合元素 $("p")
* 匹配所有元素 集合元素 $("*")
selector1, selector2 … 将每个选择器匹配到的元素合并到一起 集合元素 $("div, span, .myClass")

层次选择器

获取后代元素,子元素,兄弟元素,相邻元素等

  1. $('body div') : body中的所有div元素
  2. $('body > div') : body的div子元素(注意只包括直接子元素,这是和前者的区别, 并不递归的包含,也就是子元素内 div元素并不被该模式识别)
  3. $('.one + div') : class为one的元素的下一个div兄弟元素,可以用 $('.one').next('div')代替
  4. $('.one ~ div') : class为one的元素后的所有div兄弟元素, 可以用 $('.one').nextAll('div')代替

DOM操作

  1. 创建节点: 使用 $(...), 比如 $('<div>some content</div>').
  2. 插入节点 $(A).xxx(B) 其中xxx代替下面的方法，注意当A是元素集合时，那么B插 入A时会将B插入每个A元素，反之亦然
    • append, appendTo : 前者是B作为最后一个子元素插入A中,后者相反,是A作为最后一个子元素插入B中
    • prepend, prependTo : 前者是B作为第一个子元素插入A中,后者相反,是A作为第一个子元素插入B中
    • after, insertAfter : 前者是B作为一个兄弟元素插入A之后,后者相反,是A作为一个兄弟元素插入B之后
    • before, insertBefore : 前者是B作为一个兄弟元素插入A之前,后者相反,是A作为一个兄弟元素插入B之前
  3. 删除节点
    • .remove() :删除节点(A.remove() 删除A)
    • .empty() :清空节点内容(A.empty() 清空A,就是将innerHTML设为'')
  4. 复制节点
    • .clone() : 复制节点，如果.clone(true)会复制绑定的事件
  5. 替换节点
    • .replaceWith: A.replaceWith(B) 用B替换A中每个元素(A可能是一个元素集合)
    • .replaceAll : A.replaceAll(B) 用A替换B中每个元素(B可能是一个元素集合)
  6. 包裹节点
    • .wrap: $('strong').wrap('<b></b>') 将所有的元素单独包裹
    • .wrapAll: 将所有匹配的元素用一个元素来包裹
  7. 属性操作
    • .attr(name[, val]) : 一个参数是获取，两个参数是设置
    • .removeAttr(name) : 删除参数指定的属性
  8. 样式操作： 样式实际就是class属性，所以可以通过.attr('class')来获得并进行操 作，注意和css区别，css指定某一项css样式比如background-color，而这里指的是 样式表中定义的类样式（css文件中以.开头样式规则）可是为了方便jquery提供了专 门的api
    • .addClass(name) : 在class属性中添加一个样式
    • .removeClass(name) :
    • .toggleClass(name) : 在属性来回的切回，如果存在那就删除，如果不存在那就添加
  9. 获得html，文本，值 （无参数是获取，一个参数是设置）
    • .html([val]) : 注意是innerHTML (设置或者获取)
    • .text([val]) : 会去掉html标签,innerText或者textContent(设置或获取)
    • .val([val]) : 表单元素的值
  10. 遍历节点
    • .children() : 所有的子节点, $(body).children() body的所有子节点
    • .next() : 紧挨着的下一个兄弟节点
    • .prev() : 紧挨着的上一个兄弟节点
    • .siblings() : 所有的兄弟节点
    • .parent() : 父节点
  11. css-DOM
    • .css(name[, val]) : 获得和设置css属性
    • .width([val]) : 获得或者设置宽度值
    • .height([val]) : 获得或者设置高度值
    • .offset() : 离窗口的距离(viewpoint), 返回{left: val1, top: val2}.
    • .position() : 获得相对于最近的设置了relative或者absolute的父节点 的偏移. 返回{left: val1, top: val2}
    • .offsetTop([val]) : 获得或设置横滚动条相对于浏览器左边界的距离
    • .offsetLeft([val]) : 获得或设置竖滚动条相对于浏览器顶部的距离

jquery事件

  1. .bind : 可以通过该方法将特定函数绑定到特定的事件上eg: 

    $('div.li').bind('click', function(){
  alert('clicked');
})

    同理可以绑定mouseover,mouseout,focus,blur等等,但因为有些事件很常用,所以jquery提供了简写方法

  2. .click
  3. .mouseover, .mouseout
  4. 合成事件: .hover 与.toggle是jquery的两个自定义事件
    • hover(enter, leave) : 鼠标进入时调用enter,鼠标离开时调用leave
    • toggle(f1, f2, … fN) : 模拟鼠标单击,第一次单击调用f1, 第N次单击调用fN,第N+1次调用f1
    • toggle() :无参数调用时不是绑定click事件,而是使元素在显示与隐藏状态切换,也就是该元素已显示,那么调用 toggle()就会隐藏该元素,如果该元素已隐藏,那么调用toggle()就会显示该元素,这和toggleClass类似

  5. 事件冒泡 当元素嵌套，而且父元素也绑定了同一事件，那么会同时触发父元素的事件处理函数，为了阻止这种行为可以用一下 代码： 

    $(function(){
  $("#sub").bind("click",function(event){
    var username = $("#username").val();  //获取元素的值
    if(username==""){     //判断值是否为空
      $("#msg").html("<p>文本框的值不能为空.</p>");  //提示信息

      event.stopPropagation();  // 阻止事件冒泡
      event.preventDefault();  //阻止默认行为 ( 表单提交，链接(a)跳转等 )
      // 如果既要阻止事件冒泡又要阻止默认行为，那么可以直接通过
      // return false; 来实现
    }
  })
})

    给回调函数添加一个event参数，并且调用event.stopPropagation()方法，这样就可以阻止事件冒泡，也就是对sub的 click不会触发sub的父元素的click事件，

  6. 模拟事件(不是用户触发，而是通过js模拟触发)
    • .trigger: \(('#btn').trigger('click) 触发click事件， 也可以直接\)('#btn').click()
    • 传递数据： .trigger(type, [.data])

jquery动画

  1. .hide([speed]), .show([speed]), .toggle([speed]) : 显示，隐藏，如果指定speed，那么会渐变的改变元素的高 度
  2. .fadeIn([speed]), fadeOut([speed]), .fadeToggle([speed]) :显示，隐藏，如果指定speed，那么会渐变的改变 元素的透明度，直到元素消失或者显示，也就是动画过程中元素的高度是不变的
  3. slideUp， slideDown，slideToggle：滑动效果

ajax

jquery ajax的回调函数的原型：

  • .load: function callback(responseText, statusText, XMLHttpRequest)
  • $.get, $.post, $.ajax: function callback(data, statusText)

其中textStatus可以是'success', 'error', 'notmodified', 'timeout'

  • .load(url [, data] [, callback]) :将返回的html插入DOM元素中, callback无论请求成功与否都会调用
  • $.get(url [, data] [[, callback] [, type]) : type可以是"html"，"xml"，"json", callback只有在请求成功 (http 200)时才调用
  • \(.post : 和\).get类似，只是通过post传送数据
  • $.getScript, $.getJSON :前者可以用来加载js文件，后者用来获取json用来给回调函数操作
  • $.ajax(options) :
  • .serialize() :在form提交时可以直接序列化form内的数据填入data域，返回字符串
  • .serializeArray() : 和serialize方法类似，不过不返回字符串，而是返回json数据
  • $.param() : $.param({a:1, b:2, c:3}) ====> a=1&b=2&c=3, 该方法是.serialize方法的核心
