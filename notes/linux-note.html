linux mint 17双显示器

主要是配置从显示器的任务栏

  1. 安装Dual Monitor Panels 扩展，记住是扩展，你应该下载文件并解压 到~/.local/share/cinnamon/extensions， 去该页面可以看到详细的配置选项。
  2. 安装Window List For Multi Monitor 1.0b，记住这是applet，所以直接内置安装

进程监控

使用upstart的respawn, 这是比较简单的情况,如果有更多的需求可以使用supervisor或 者monit, 二者都有一些缺陷,比如supervisor要求程序必须前台运行,monit要求每一个程 序都要创建pidfile,如果你要根据资源占用来监控,那么monit是唯一的选择

wireshark的配置

  1. sudo dpkg-reconfigure wireshark-common, 图形界面点yes
  2. sudo adduser $USER wireshark
  3. reboot

命令行工具

strace

跟踪程序的系统API调用，对调试程序很有帮助。

htop

可以以一种图形化的方式显示系统的各项资源占用

ifstat

简单的监控网络流量

lsof

list open files， 显示哪些进程打开了哪些文件。

ssh

ssh要认证，必须将本地公钥上传到服务器的authorized_keys文件中，并且该文件必须是 600或者700权限。（不能用660,664这样的权限，只能是600或者700）

ssh免密码登录

  1. 先生成密钥：=ssh-keygen -t rsa= 所有的地方都enter
  2. 将公钥上传到目标服务器： ssh-copy-id -i /root/.ssh/id_rsa.pub root@remote_host, 也可以手动上传但是要注意权限

给力的工具

  1. zsh： 非常给力的shell，比bash好用
  2. percol： 可以增量式的过滤shell命令的输出结果，可以和zsh结合定制出ppkill， ppgrep命令
  3. autojump： 可以非常快速的跳转到你常用的目录，有zsh插件
  4. cheat： 对常用的linux命令的cheatsheet，很多时候比man要好用
  5. cloc: 统计代码行数
  6. tree： 显示树形的目录结构，很直观
